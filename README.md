# 自然选择模拟器 🐰

基于PhET开源项目改编的生物学自然选择模拟器，专门为教育用途设计。

## 🌟 项目特色

- **交互式模拟**: 实时观察兔子群体在不同环境下的进化过程
- **多种环境**: 草原、雪地、沙漠三种环境类型
- **捕食者压力**: 可选择不同的捕食者来施加选择压力
- **遗传机制**: 包含突变、遗传和自然选择的完整机制
- **实时统计**: 动态显示群体特征分布和世代变化
- **响应式设计**: 支持桌面和移动设备

## 🔬 模拟机制

### 兔子特征
- **棕色毛皮**: 在草原环境中有伪装优势
- **白色毛皮**: 在雪地环境中不易被发现
- **金色毛皮**: 稀有突变特征，在沙漠环境中有优势

### 环境因素
- **草原环境**: 绿色背景，有利于棕色毛皮
- **雪地环境**: 白色背景，白色毛皮更容易生存
- **沙漠环境**: 沙色背景，金色毛皮适应性最强

### 选择压力
- **鹰类捕食**: 主要影响在非适应环境中显眼的个体
- **狼群捕食**: 对群体施加整体生存压力
- **无捕食者**: 观察没有外部压力时的群体动态

## 📊 教育价值

这个模拟器帮助学生理解：
- 自然选择的基本原理
- 环境压力如何影响群体特征
- 遗传变异的重要性
- 适应性特征的进化过程
- 群体遗传学的基础概念
